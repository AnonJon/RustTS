# Audio Assets

Place `.ogg` audio files here. The game will automatically load them if present.

## Expected Files

| File | Description |
|------|-------------|
| `select.ogg` | Unit selection acknowledgment |
| `command.ogg` | Unit move/command acknowledgment |
| `attack.ogg` | Weapon/combat sound |
| `build.ogg` | Construction hammer/building sound |
| `complete.ogg` | Building construction complete |
| `gather.ogg` | Resource gathering (chopping, mining) |
| `death.ogg` | Unit death sound |
| `age_up.ogg` | Age advance fanfare |

All files are optional - missing sounds are silently skipped.
